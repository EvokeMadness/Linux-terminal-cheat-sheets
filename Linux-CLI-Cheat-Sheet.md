# Linux CLI Cheat Sheet

Create a new user, 'allie' and give them a "home" directory
```
useradd -m allie
```

Create/Change a user's password
```
passwd allie
```

Display the current location in the Linux directory
```
pwd
```

Move to a location below the current location in the directory, 'Documents/'
```
cd Documents/
```

Move to the location directly above the current location (move to the parent directory)
```
cd ..
```

Move to a specific location in the Linux directory, '[directory name]/[directory name]/'
```
cd /home/allie/
```

Move to the user's home directory
```
cd ~
```
